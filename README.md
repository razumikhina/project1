
#проект для стажировки QA

проверка API и автотесты на Python.

Что делаю
1.Отправляю запросы к API jsonplaceholder.typicode.com
2.проверяю, что приходят имя и email
3.Пишу автотесты с pytest

технологии
1.Python
2.requests
3.pytest
4.Git

как запустить
pip install requests pytest
python src/api_checker.py
pytest tests/
